# 📡 Telco Customer Churn Predictor

End-to-end ML pipeline for predicting telecom customer churn. Optimized for recall to minimize false negatives (business cost-aware).

## 🚀 Live Demo
[![Hugging Face Spaces](https://img.shields.io/badge/🤗_HF_Spaces-Live_Demo-blue)](https://pandrive786-telco-churn-predictor.hf.space/)

## 📊 Model Performance
| Metric | Value |
|--------|-------|
| ROC-AUC | 0.836 |
| Recall (Churn) | 0.789 |
| Precision (Churn) | 0.612 |
| Optimization | Optuna (Hyperparameter Tuning) |

## 🛠️ Tech Stack
- Python, pandas, scikit-learn
- Optuna (Auto-Tuning), joblib (Serialization)
- Gradio (UI), Hugging Face Spaces (Deployment)
- GitHub Actions (CI/CD)
